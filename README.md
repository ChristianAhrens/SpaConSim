# SoundscapeOSCSim
![SoundscapeOSCSim Logo](/Resources/Images/SoundscapeOSCSim.png)

## This a minimal tool to simulate a d&b audiotechnik DS100 Soundscape Processor

It utilizes a RemoteProtocolBridge processing node with fixed parametrization for core functionality.
See https://github.com/ChristianAhrens/RemoteProtocolBridge for details.
